In the course Variables we explained how variables are used to store data values. We mainly addressed the data types Strings and numbers. Besides these data types, JavaScript also has a data type called Arrays. Arrays are a special kind of data type because it is a set of values, not just a single value. The syntax of an array is as follows:

var nameOfArray = [arrayValue1, arrayValue2, ...];

For example, if we want to store fruit names in an array, it would look something like this:

var fruit = ["banana", "apple", "orange"];

Arrays are useful, because we can use them to store multiple values under one variable name. You can access the value of an array by referring to the index number of the value. The first value in an array starts at index number 0. The second value is index 1, the third value is index 2, etc. Using this index number, we can access values by using the following syntax:

var valueInArray = nameOfArray[indexNumber]

Following up on the fruit example, you can access the value "apple" by using it's index:

var apple = fruit[1];

Arrays are commonly used in combination with for loops. This is because the variable which determines the incrementation of the for loop (for example: var x) can be used as the index number of an array. Let's do this for the fruit array as well:

var fruit = ["banana", "apple", "orange"];
for ( var x = 0; x < fruit.length; x++ ) {
john.say(fruit[x]);
}

The example above uses a for loop to cycle through all values of the array, and makes John say the values of fruit. Notice how the length of the array is used to define the maximum amount of iterations the for loop can make (x < fruit.length).

Note 1: You can store different data types in the same array. The following example is a properly working array:

var randomValues = ["Some text", 12345];

Note 2: You can use names to access values in an array. This can be done as follows:

var carArray = [brand: "BMW", wheels: 4, passengers: 4];
var carBrand = carArray[brand];

This method is an elaborate way of accessing a value in an array, but it adds extra clarity on how to access the values.

Note 3: If you want to add new values to an existing array, you can do so with the push() action:

var fruit = ["banana", "apple", "orange"];
fruit.push("lemon");
// The new array is now ["banana", "apple", "orange", "lemon"]
