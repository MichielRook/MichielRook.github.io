Properties are characteristics of objects in JavaScript. You can compare them with body features of a person. In the sentence "John is a 25 year old man." we can picture a general idea of who John is; his age is 25 and his gender is male. The same way we can tell the computer what John's body features are, using properties. For example:

var john = {
name: "John",
age: 25,
gender: "male"
};

i.say(john.name + " is a " + john.age + " year old " + john.gender + ".");

The example above makes the i object say exactly the same sentence we used as an example for John.

Properties are often used in situations where you're working with different objects that share a variable, but have different values for those variables. For example, John is 25 years old while Sarah is 23 years old. They both share the variable age but have a different value stored in it. Therefore, age coulb be used as a property, with different values for both John and Sarah.

Note 1: You can add properties after your first definition of the object. You can do this by creating a new property name and assigning a value to it:

object.newPropertyName = value ;
