In programming, you sometimes want to take different actions according to the situation. This means that you need to be able to tell the computer to check what the situation is. This can be done by using if statements.

An if statement checks whether a condition is true or false (just like the while & for loop). If (see the transition from human language to Javascript?) the condition is true, the code included in the if statement will be executed. Otherwise, this code will be skipped. In general, an if statement looks like this:

if (condition ) {
// Some code goes here
}

In most scenario's, you would like the computer to perform a different action if the condition is false, not skip code entirely. You're offering the computer an alternative, so to speak. This alternative is called the else statement. If we combine this with the if statement, it will look like this:

if (condition ) {
// The if code goes here
} else {
// The else code goes here
}

The above example gives you the oppurtunity to let the computer check for two possible situations. Sometimes, however, you would like the computer to check between more than two situations. You can do this using an else if statement:

if (condition 1 ) {
// Condition 1 is true, code comes here
} else if (condition 2 ) {
// Condition 1 is false and condition 2 is true, code comes here
} else {
// Condition 1 and 2 are false, code comes here
}

Let's take a look at a comprehensive example. While ordering a pizza, you want to be able to choose between multiple pizzas. We can translate these options to an if statement:

if (pizzaTonno ) {
// Buy pizza Tonno
} else if (pizzaMozzarella ) {
// Buy pizza Mozzarella
} else if (pizzaHawaii ) {
// Buy pizza Hawaii
} else if (pizzaFunghi ) {
// Buy pizza Funghi
} else {
// Buy pizza Margherita
}

Note 1: You can use many else if statements if you want to, but using too many will slow down the computer drastically.

Note 2: Once an (else) if statement equals true, the rest of the statements will not be evaluated anymore. Therefore, keep in mind that the (else) if statements should be sorted by importance in descending order.

Exercise: Create a variable myName and store your name in it. Then, check if myName is equal to "John". If it is, use say() to say: "I am John!". Otherwise, use say() to say: "I am not John!".
