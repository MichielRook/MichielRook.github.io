An alternative to the while loop is the for loop. Essentially, a for loop is most commonly used in a scenario where you know how many times you wish to iterate through a certain piece of code. The for loop syntax is as follows:

for (statement 1; statement 2; statement 3 ) {
//piece of code to execute
}

for is the keyword that tells the computer you are creating a for loop. Followed by the keyword is a set of statements. Statement 1 represents the initiation of a variable. Statement 2 represents the condition (similar to the while loop). Statement 3 represents the incrementation of the variable. Following up on the example used in the last course, While-loops, we get the following for loop:

for (var x = 0; x < 5; x++) {
i.moveRight();
}

Note 1: The variable used in the first statement can be initiated outside the for loop. The for loop would then look like this:

var x = 0;
for ( ; statement 2; statement 3 ) {
//piece of code to execute
}

Note 2: The value of the variable can also be incremented inside the for loop. In that case, the for loop will look like this:

for (statement 1; statement 2; ) {
//piece of code to execute
x++;
}

Note 3: You can combine both notations from Note 1 and Note 2. The final result would be:

var x = 0;
for ( ; x < 5; ) {
//piece of code to execute
x++;
}

Note 4: You can prematurely stop a for loop by using the break keyword:

for (var x = 0; x < 5; x++) {
i.say(x);
break;
}

The example above will only execute one iteration of the for loop, after which it will stop iterating.

Note 5: It is possible to create a for loop inside a for loop. This is called a nested loop. Nested loops are used to iterate through iterations. Take a look at the following example:

for (var x = 0; x < 10; x++) {
for (var y = 0; y < 10; y++) {
i.say("Hi!");
}
}

This example iterates through x 10 times (0-9), and iterates through y 10 times per iteration of x. As a result, i.say("Hi!") results in the i object saying "Hi!" 100 times.
