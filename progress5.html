A while loop is a construction in programming, which allows the computer to repeat a certain piece of code. Every while loop should look something like this:

while (condition) {
//piece of code to execute
}

while is the keyword to tell the computer you are creating a while loop. The condition determines if the while loop is executed; if the condition is correct (true), the code will execute. However, if the condition is not correct (false), the while loop will not start running. Keep in mind that the condition should be put between parentheses: ( ).

Followed by the keyword and condition, are the curly brackets. better known as accolades: { }. Any line of code written between the accolades will be looped through when the condition is true.

Keep in mind that in order for your code to escape the while loop, you need to adjust the condition within the while loop itself. Otherwise, a so called infinite loop will arise; a loop which will never end. For example:

while (x < 5) {
i.moveRight();
x++;
}

If the code x++; above were to be left out, the value of x would not increase, and therefore result in an infinite loop. Sometimes an infinite loop is desirable. In that case you can use true as the condition, as true is always true. This will create an infinite loop.

Note 1: Keep in mind that the variable used in the condition needs to be initiated before creating the while loop.
