In the previous courses, we used the example i.moveTo("John");. As you have probably noticed, John is written between quotation marks. This is because John is a so called String.

A String is a combination of characters which together compose a name or a sentence. It's important to put the String between quotation marks, or the computer won't recognize it as a String. You can use both double (") and single (') quotation marks to tell the computer something is a String.

Keep in kind that you have to be consistent in using double or single quotation marks. For example: 'This is a String" will not be recognized as a String!

Note: If you want to use the action say() to say a number, you have to say it as a String! 5 is a number, "5" is a String!

Exercise: Change the moveTo() action in the editor to the left, so that the computer recognizes John as a String. Also, use the say() action to say: Hi John!

// Change this line of code so John becomes a String i.moveTo(John); // Use the say() action to say: Hi John!
