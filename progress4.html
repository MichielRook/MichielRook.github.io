Variables are used to hold data values. There are different reasons on why to use them, but the most commonly used reason is to save yourself some time. Let's say we are writing a computer program to calulate all sorts of values with the mathematical number Pi .

The value of Pi, showing the first 10 digits, is 3.1415926535 . Imagine how many times you would have to repeat all these numbers if you have to do 3 or even more calculations using Pi!

So, how can we use variables to make our lives easier? It's very simple! All we have to do is tell JavaScript we are creating a variable, give it a name and assign a value to it! Following up on our example of Pi, it would look something like this:

var pi = 3.1415926535;

The word var tells JavaScript we are creating a variable, and = indicates we are assigning a value to it.

Now that we have created a variable, it's important to know how to use it. Supposing we want to tell John the value of Pi, we could write: i.say("3.1415926535"); . Using the variable example of pi, however, we could get the same result by writing:

i.say(pi);

By using the name we gave our variable (pi), the computer understands that we want to tell the value stored inside of it to John!

In JavaScript, we can store every form of data values inside a variable. This makes using variables very convenient!

Note 1: You can also create a variable first, and assign the value later. This would look something like this:

var name;
name = "John";

Note 2: If you want to combine a variable with a String, you have to add the variable to the String using the + symbol:

var word = "String";
i.say("This is a " + word);

Note 3: Keep in mind that while accessing the value of a variable, you do not use quotation marks to refer to it's name!
